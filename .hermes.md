# MCP Remnawave

Hermes: when working in this repo, follow these rules.

## Git
- Push to **origin** (mitetenov/mcp-remnawave), NEVER to upstream (TrackLine/mcp-remnawave)
- Create PRs against origin's main branch only
- Use `gh` CLI, not raw git for PRs

## Build
- `npm run build` before committing
- No `npm install` without changes to package.json

## Docker
- Image: `mitetenov/remnawave-mcp` on Docker Hub
- When bumping version: update package.json → CI auto-releases + builds image
